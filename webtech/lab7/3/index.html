Resolution

Resolution is a simple count of the number of pixels across the entire width and height of a device. The Samsung Galaxy Note 8 resolution is 2960x1440. Compare that to the Samsung Galaxy Note II which is 720x1280.



Flexible

A grid that is flexible, which automatically resizes itself according to the size of the window.

Media query

Media queries allow the browser to use different styling depending on certain conditions. This functionality is used in RWD to modify the layout at specific widths in the browser, also called “breakpoints”.

Flexible media

Flexible images allow the browser to scale images depending on the size of the browser.

Adaptive Website

Adaptive design detects specific devices to provide the user with a separate mobile website. Like responsive design, it also aims to optimize the user’s viewing experience.
